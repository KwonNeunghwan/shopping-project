#KDT풀스택 2차 프로젝트
# Shoppingmall project

> ## Shoppingmall Project 기능

---

- 기본페이지
- 로그인 페이지, 회원가입 페이지
- 아이디 비번찾기
- 마이페이지
- 리뷰기능
- 관리자 로그인(상품등록, 상품목록, 회원목록)


  

> ## 추가적으로 구현할 기능
- 장바구니
- 주문 테이블
